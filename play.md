# Curso de NestJs

## Docs

https://docs.nestjs.com/

### Iniciando uma Aplicação em Nest

#### Instalação

    npm i -g @nestjs/cli

#### Iniciando Um Projeto

    nest new project-name


#### Criando um Módulo

    nest g module nome-modulo



#### Criando um Controller

    nest g controller nome-controller    



#### Criando um Service

    nest g service nome-service
